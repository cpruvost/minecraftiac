# Comment installer de manière automatisée un serveur Minecraft dans Oracle OCI

Pour réaliser ce tuto, vous avez besoin en termes de pré-requis :

- d'un compte Oracle Cloud Infrastructure (FREE ou pas) et de connaissances sur comment créer un VM.
- d'un serveur Jenkins opérationnel
- d'un serveur Vault (Hashipcorp) opérationnel
- de connaître Terraform (Hashicorp), Ansible, et Linux
